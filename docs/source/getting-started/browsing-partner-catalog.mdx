# Browse the partner catalog

## Overview

The **NKP Partner Catalog** contains validated third-party applications from Nutanix technology partners. These applications have been integrated and tested for deployment on NKP through the partner ecosystem.

**Repository:** [nutanix-cloud-native/nkp-partner-catalog](https://github.com/nutanix-cloud-native/nkp-partner-catalog)

Browse the repository's `applications/` directory for the current list of available applications and their versions.

## Characteristics

- Require **NKP Pro** or **NKP Ultimate** license
- Some applications may require a separate partner license in addition to the NKP license
- Deploy at **workspace** scope
- Applications span categories such as API gateway, backup and DR, virtual desktops, and more

## Deploying from the catalog

1. Log into the NKP UI and navigate to your workspace.
2. Go to **Applications** in the sidebar.
3. Browse applications under the Partner catalog section.
4. Click **Enable** and configure required values (e.g. license keys, domain names).

## Contributing a partner application

Partner applications follow the same catalog structure as all NKP catalog applications. See the [Workflows](/docs/workflows/initialize-catalog-application) section for adding applications to a catalog repository.
