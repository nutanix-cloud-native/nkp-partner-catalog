# Browse the Nutanix product catalog

## Overview

The **Nutanix Product Catalog** contains Nutanix-supported applications that extend NKP with capabilities such as AI inference, networking, observability, and data services. These are first-party applications validated and certified by Nutanix.

**Repository:** [nutanix-cloud-native/nkp-nutanix-product-catalog](https://github.com/nutanix-cloud-native/nkp-nutanix-product-catalog)

Browse the repository's `applications/` directory for the current list of available applications and their versions.

## Characteristics

All applications in this catalog:

- Require **NKP Pro** or **NKP Ultimate** license
- Are **Nutanix-supported** with official certification
- Support **airgapped** deployments
- Deploy at **workspace** scope

## Deploying from the catalog

1. Log into the NKP UI and navigate to your workspace.
2. Go to **Applications** in the sidebar.
3. Find the application under the Nutanix catalog section.
4. Click **Enable** and configure any required values.
