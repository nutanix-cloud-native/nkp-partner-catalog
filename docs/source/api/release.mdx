# CatalogReleaseSpec schema

JSON Schema for **CatalogReleaseSpec**: the catalog release specification that maps NKP versions to supported applications. Each release defines a catalog collection (e.g. tag, file name, airgapped flag) and either **constraints** (filter by NKP version) or an explicit **applications** list.

**Schema file:** [catalogreleasespec.schema.json](/schemas/v1/catalogreleasespec.schema.json)
